## How to start

1. redis를 실행시킵니다

>

    $ redis-server

2. server를 실행시킵니다.

>

    $ cd server
    $ yarn dev

3. web을 실행시킵니다.

>

    $ cd web
    $ yarn dev

 
그렇다면 web 및 서버는 다음과 같은 주소에 형성됩니다.
>

    web: localhost:3000
    server: localhost:4000/graphql
